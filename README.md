# 💬 Gerador de Templates WhatsApp

> Uma ferramenta web leve, rápida e 100% responsiva para gerar mensagens formatadas para o WhatsApp. Ideal para automação de cobranças, renovações de planos, avisos de suspensão e boas-vindas de clientes.

🔗 **Acesse a versão online agora:**  
👉 [https://rubenbruno89.github.io/gerador-templates/](https://rubenbruno89.github.io/gerador-templates/)

---

## ✨ Funcionalidades

- 📱 **100% Responsivo**: Layout adaptável que funciona perfeitamente em celulares, tablets e desktops.
- 🎨 **5 Modelos Prontos**: 
  - 📅 Lembrete de Renovação (3 dias antes)
  - ⏳ Aviso de Vencimento (Hoje)
  - ⚠️ Aviso de Suspensão
  - ✅ Confirmação de Pagamento
  - 🎉 Bem-vindo / Novo Cadastro
- 👁️ **Preview em Tempo Real**: Visualize exatamente como a mensagem ficará no WhatsApp (com *negrito* e _itálico_).
- 📋 **Cópia com 1 Clique**: Botões dedicados para copiar o texto formatado ou o texto bruto.
- 🌙 **Dark Mode Automático**: Respeita a preferência de tema do sistema operacional do usuário.
- ♿ **Acessibilidade**: Navegação por teclado, foco visível e respeito à configuração de "redução de movimento".
- ⚡ **Zero Dependências**: Feito apenas com HTML5, CSS3 e JavaScript puro (Vanilla). Não requer backend nem instalação.

---

## 🚀 Como Usar

1. Acesse o link: [https://rubenbruno89.github.io/gerador-templates/](https://rubenbruno89.github.io/gerador-templates/)
2. Selecione o **Template** desejado na parte superior.
3. Preencha os campos: **Nome**, **Tipo/Produto**, **Data de Vencimento**, **Valor** e **Chave PIX**.
